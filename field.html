Research Fields


Gametech

Divisi Gametech (Game Technology) berfokus pada pengembangan teknologi game. Divisi ini terbagi menjadi 3 tim yaitu: programmer, asset design dan sound design. Selain melakukan riset, divisi Gametech juga aktif mengikuti berbagai perlombaan seperti PUSPITEK MOGRAPH dan GEMASTIK


GIS

Divisi GIS (Geographic Information System) berfokus pada pengembangan teknologi aplikasi mobile dan UI/UX, 3d modelling dan augmented reality (AR) dan remote sensing. Selain itu divisi GIS juga rutin melaksanakan pengabdian masyarakat setiap tahunnya.


Big Data

Divisi Big Data berfokus pada bidang data analyst, data engineer, data scientist dan pengembangan machine learning. Kegiatan yang dilakukan oleh divisi Big Data antara lain meriset serta menganalisis berbagai data dan mengikuti berbagai perlombaan seperti Hackathon.


Cyber Security

Divisi Cyber Security berfokus pada pengembangan keamanan siber dan Dev ops. Divisi ini terbagi menjadi 3 tim yaitu: Blue Team, Red Team dan Cloud Team. Divisi ini juga mengikuti berbagai perlombaan seperti CTF dan GEMASTIK
